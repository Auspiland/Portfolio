Self-hosted Multi-Agent System

집에 둔 미니 서버 위에서 도는, 역할이 분리된 멀티 에이전트 시스템.

manager · reader · worker · auditor 네 역할로 작업을 분리해, 한 에이전트가 계획하고 다른 에이전트가 실행·검증하도록 설계했습니다. Claude API로 추론하고, 이상 징후는 텔레그램 봇으로 즉시 알립니다. 전용 시스템 유저와 Docker 샌드박스 위에서 동작하며, 방화벽·무결성 검사·접근 격리 등 다단계 보안 하드닝을 적용했습니다.
